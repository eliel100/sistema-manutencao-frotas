# Sistema de Manutenção de Frotas de Veículos

Projeto desenvolvido para a disciplina de Programação para Web.

## Descrição

O sistema permite gerenciar veículos, tipos de serviço e manutenções realizadas em uma frota de veículos.

A aplicação possui frontend em React com Vite, backend em Node.js com Express e banco de dados PostgreSQL.

## Tecnologias utilizadas

- React
- Vite
- Bootstrap
- Axios
- Node.js
- Express
- Sequelize
- PostgreSQL
- PWA
- Render
- Vercel

## Funcionalidades

- CRUD de veículos
- CRUD de tipos de serviço
- CRUD de manutenções
- Relacionamento entre tabelas
- Frontend consumindo API
- Aplicação PWA
- Deploy em nuvem
